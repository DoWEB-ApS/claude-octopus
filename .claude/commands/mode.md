# Mode

Manage enterprise execution mode in `.doweb/policy/project-run.yaml`.

## Usage

```bash
${CLAUDE_PLUGIN_ROOT}/scripts/orchestrate.sh mode
${CLAUDE_PLUGIN_ROOT}/scripts/orchestrate.sh mode supervised
${CLAUDE_PLUGIN_ROOT}/scripts/orchestrate.sh mode autonomous
```
